# 📊 Microsoft Stock Analysis Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive **Microsoft (MSFT) Stock Market Dashboard** built in **Power BI**. The dashboard analyzes historical stock price data and provides insights into price movement, trading volume, market trends, and overall stock performance using interactive visualizations.

The objective of this project is to transform raw stock market data into meaningful business insights that help users understand historical trends and make data-driven observations.

---

# 🎯 Objectives

* Analyze Microsoft's historical stock performance.
* Track daily and long-term price trends.
* Monitor trading volume over time.
* Compare Open, High, Low, and Close prices.
* Create an interactive dashboard for quick business insights.

---

# 📈 Dashboard Features

The dashboard includes interactive visualizations such as:

* KPI Cards

  * Latest Closing Price
  * Highest Price
  * Lowest Price
  * Average Closing Price
  * Total Trading Volume

* Line Chart

  * Closing Price Trend

* Candlestick / Stock Price Visualization *(if available)*

* Volume Trend Chart

* Monthly / Yearly Performance Analysis

* Date Slicer

* Interactive Filters
